Difference between justify-items and justify-self

In CSS Grid Layout, justify-items and justify-self are properties used to control the horizontal alignment (along the inline axis) of grid items within a grid container, they are applied at different levels and have distinct purposes. justify-items is applied to the grid container and defines the default horizontal alignment for all grid items within the grid. It affects all the grid items uniformly, setting their horizontal alignment within their respective grid cells.
